NParkBuddy Server

Welcome to the NParkBuddy server!

Code for the NParkBuddy server can be found here.

Endpoints for the NParkBuddy server are as follows:

Endpoints Summary
Method Endpoint Description Example
GET / Root endpoint
POST /profiles/create Creates a new user account>
POST /profiles/login Logs in a user
POST /profiles/<username>/change_password Changes a user's password
GET /parks Gets all parks
GET /parks/<park_name> Gets a park by name Example:/parks/East_Coast_Park
POST /bookings Creates a booking
GET /timeslots/<park_name>/<facility_name>/<date> Gets available timeslots Example:/timeslots/East_Coast_Park/BBQ_Pit_47/23-May-2025
GET /weather Gets weather warning
GET /profiles/<username>/bookings Gets all bookings by profile Example:/profiles/nparkadmin/bookings
POST /bookings/cancel Cancels a booking
POST /reviews Reviews a booking
GET /facilities Gets all facilities
GET /facilities/filter Filters facilities Example:/facilities/filter?type=BBQ_pit
GET /reviews/<park_name>/<facility_name> Views reviews Example:/reviews/East_Coast_Park/BBQ_Pit_47
POST /profiles/<username>/change_username Changes a user's username
POST /profiles/<username>/change_email Changes a user's email
GET /log-viewer View server logs
